تعليم CSS بالعراقي

درس 1: مقدمة عن CSS

CSS (توزيع الأنماط المتعاقبة) تستخدم لتنسيق مظهر صفحات الويب. تسمح لك بتغيير الألوان، الخطوط، والتخطيطات.

درس 2: كيفية ربط CSS بالـ HTML

يمكنك ربط CSS بملف HTML باستخدام وسم <link> في قسم <head>:

<link rel="stylesheet" href="style.css">

درس 3: تحديد الألوان

يمكنك تحديد الألوان باستخدام الأسماء، الأكواد السداسية، أو القيم RGB. مثال:

color: #4CAF50;

درس 4: الخطوط

يمكن تغيير الخط باستخدام خاصية font-family. مثال:

font-family: 'Arial', sans-serif;

درس 5: تحديد الأحجام

يمكنك تحديد حجم الخط باستخدام font-size. مثال:

font-size: 16px;

درس 6: الهوامش والحشوات

يمكن التحكم في الهوامش باستخدام margin والحشوات باستخدام padding. مثال:

margin: 10px; padding: 20px;

درس 7: الخلفيات

يمكنك تعيين خلفيات للعناصر باستخدام خاصية background. مثال:

background-color: lightblue;

درس 8: الفلاتر والظل

يمكن إضافة تأثيرات الظل باستخدام خاصية box-shadow. مثال:

box-shadow: 2px 2px 5px #aaa;

درس 9: قوائم التنقل

تستخدم قوائم التنقل لتوجيه الزوار بين الصفحات. مثال على قائمة بسيطة:



    
    
  • الرئيسية
    • 
    
  • عن الموقع
    • 
    
  • تواصل معنا
    •

درس 10: تصميم الزر

يمكن تصميم الأزرار باستخدام CSS. مثال:


button {
    background-color: #4CAF50;
    color: white;
    border: none;
    padding: 10px 20px;
    cursor: pointer;
}

درس 11: التصميم المتجاوب

يساعد التصميم المتجاوب في جعل الموقع يتناسب مع جميع أحجام الشاشات باستخدام @media.


@media (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

درس 12: الأنيميشن في CSS

يمكن إضافة الحركة للعناصر باستخدام خاصية animation. مثال:


@keyframes mymove {
    from {top: 0px;}
    to {top: 200px;}
}

.animated {
    position: relative;
    animation: mymove 2s;
}

درس 13: استخدام Flexbox

Flexbox يجعل من السهل تنظيم العناصر في صفوف أو أعمدة. مثال:


.container {
    display: flex;
}

درس 14: استخدام Grid Layout

Grid Layout يوفر نظام شبكي لتنسيق العناصر بشكل احترافي. مثال:


.grid-container {
    display: grid;
    grid-template-columns: auto auto auto;
}

درس 15: أدوات التطوير في المتصفح

تساعد أدوات التطوير على فحص العناصر وتعديلها مباشرة في المتصفح.

درس 16: الشفافية

يمكن تعديل شفافية العناصر باستخدام خاصية opacity. مثال:

opacity: 0.5;

درس 17: استخدام الصور

يمكن تخصيص الصور باستخدام خاصية background-image. مثال:


background-image: url('image.jpg');

درس 18: تصميم بطاقات

تستخدم البطاقة لتجميع المحتوى بشكل منظم. مثال:


.card {
    border: 1px solid #ddd;
    border-radius: 5px;
    padding: 15px;
    margin: 10px;
}

درس 19: استجابة التصميم

تصميم يستجيب لمقاسات الشاشات المختلفة باستخدام وحدات مرنة مثل النسبة المئوية.

درس 20: استخدام المتغيرات في CSS

يمكن استخدام المتغيرات لتسهيل إدارة الألوان والخطوط. مثال:


:root {
    --main-color: #4CAF50;
}

body {
    background-color: var(--main-color);
}
