مقدمة الدروس

مقدمة

HTML هي لغة ترميز تستخدم لإنشاء صفحات الويب. تعتبر أساسية لتصميم المواقع الإلكترونية، وتساعدك على هيكلة المحتوى بطريقة مفهومة.

درس 1: مقدمة عن HTML

في هذا الدرس، سنتعرف على مفهوم HTML ولماذا هي مهمة في تطوير الويب.

درس 2: بنية HTML

تتكون صفحات HTML من عناصر وعلامات. يجب أن تبدأ كل صفحة بـ <!DOCTYPE html>، يليها عنصر <html> الذي يحتوي على رأس <head> وجسم <body>.

<!DOCTYPE html>
<html>
    <head>
        <title>عنوان الصفحة</title>
    </head>
    <body>
        محتوى الصفحة هنا
    </body>
</html>

درس 3: علامات HTML الأساسية

هناك العديد من العلامات المهمة في HTML، مثل:

  • <p>: لكتابة فقرة.
  • <a>: لإنشاء روابط.
  • <img>: لإضافة صور.
  • <div>: لتقسيم المحتوى.
  • <h1> إلى <h6>: لتحديد العناوين.

درس 4: الروابط

يمكنك إنشاء روابط باستخدام علامة <a> كالتالي:

<a href="https://www.example.com">اذهب إلى Example</a>

درس 5: الصور

لإضافة صورة، استخدم علامة <img>:

<img src="رابط_الصورة.jpg" alt="وصف الصورة">

درس 6: القوائم

يمكنك إنشاء قوائم باستخدام العلامات التالية:

<ul> ... </ul>  // قائمة غير مرتبة
<ol> ... </ol>  // قائمة مرتبة

درس 7: الجداول

لإنشاء جدول، استخدم العلامات التالية:

<table>
    <tr>
        <th>عنوان العمود</th>
    </tr>
    <tr>
        <td>بيانات</td>
    </tr>
</table>

درس 8: النماذج

لإنشاء نموذج، يمكنك استخدام العلامات التالية:

<form>
    <input type="text" placeholder="ادخل اسمك">
    <input type="submit" value="إرسال">
</form>

درس 9: العناوين

تستخدم العلامات <h1> إلى <h6> لتحديد العناوين المختلفة في الصفحة:

<h1>عنوان رئيسي</h1>

درس 10: التعليقات

يمكنك إضافة تعليقات داخل HTML باستخدام:

<!-- هذا تعليق -->

لن يتم عرض التعليقات في المتصفح، لكنها مفيدة لكتابة ملاحظات في الكود.
