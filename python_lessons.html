موقع تعليم بايثون

مقدمة الدروس

مقدمة

بايثون هي لغة برمجة شائعة وسهلة التعلم، تستخدم في تطوير الويب، وتحليل البيانات، والذكاء الاصطناعي، وغيرها.

درس 1: مقدمة عن بايثون

بايثون لغة عالية المستوى، تتميز ببساطتها ووضوحها. تم إنشاؤها في عام 1991، وتستخدم على نطاق واسع في العديد من المجالات.

درس 2: تثبيت بايثون

يمكنك تحميل بايثون من الموقع الرسمي python.org، ثم اتبع التعليمات للتثبيت.

درس 3: أنواع البيانات

تتضمن أنواع البيانات الأساسية في بايثون:

  • أعداد صحيحة (int)
  • أعداد عشرية (float)
  • نصوص (str)
  • قوائم (list)

درس 4: المتغيرات

يمكنك تعريف المتغيرات في بايثون بسهولة. مثال:

x = 5

درس 5: العمليات الحسابية

بايثون يدعم العديد من العمليات الحسابية، مثل:

جمع: +  |  طرح: -  |  ضرب: *  |  قسمة: /

درس 6: الجمل الشرطية

تستخدم الجمل الشرطية للتحقق من شروط معينة. مثال:

if x > 10:
    print("x أكبر من 10")

درس 7: الحلقات التكرارية

يمكنك استخدام الحلقات لتكرار كود معين. مثال على حلقة for:

for i in range(5):
    print(i)

درس 8: الدوال

الدوال تستخدم لتنظيم الكود. يمكنك تعريف دالة كالتالي:

def greet():
    print("مرحبا بكم في بايثون!")

درس 9: القوائم

القوائم تستخدم لتخزين مجموعة من القيم. مثال:

fruits = ["تفاح", "موز", "برتقال"]

درس 10: القواميس

القواميس تخزن البيانات بشكل أزواج من المفاتيح والقيم:

person = {"name": "علي", "age": 25}

درس 11: المجموعات

المجموعات تستخدم لتخزين قيم فريدة:

my_set = {1, 2, 3}

درس 12: استثناءات

يمكنك معالجة الأخطاء باستخدام الجمل التالية:

try:
    x = 1 / 0
except ZeroDivisionError:
    print("لا يمكن القسمة على صفر!")

درس 13: الملفات

يمكنك فتح وقراءة الملفات بسهولة. مثال:

with open("file.txt", "r") as file:
    content = file.read()

درس 14: البرمجة الكائنية

تستخدم البرمجة الكائنية لتنظيم الكود حول الكائنات. مثال:

class Dog:
    def bark(self):
        print("الكلب ينبح!")

درس 15: مكتبات بايثون

يمكنك استخدام مكتبات خارجية لإضافة ميزات جديدة. مثال:

import math
print(math.sqrt(16))

درس 16: الواجهات الرسومية

يمكنك إنشاء واجهات رسومية باستخدام مكتبات مثل Tkinter. مثال:

import tkinter as tk
root = tk.Tk()
root.title("نافذة بايثون")
root.mainloop()

درس 17: التعامل مع البيانات

يمكنك استخدام مكتبة Pandas للتعامل مع البيانات. مثال:

import pandas as pd
data = pd.read_csv("data.csv")

درس 18: تحليل البيانات

يمكنك استخدام مكتبة NumPy لتحليل البيانات. مثال:

import numpy as np
arr = np.array([1, 2, 3])

  • برنامج آلة حاسبة

  • موقع ويب باستخدام Flask
